# The First Step into the Web Dev World: Personal Portfolio

This is my debut web project, originally developed as part of the **Fundamentals of Web Design** course during my freshman year. Built using **HTML, CSS, and JavaScript**, the project serves as a foundation for my web development journey. 

I am continuously refactoring and enhancing it to reflect my evolving technical skills and best practices.

## 🛠️ Tech Stack
* **HTML5** - Markup and structure
* **CSS3** - Styling and layout
* **JavaScript** - Interactivity and logic

## 🚀 Future Enhancements
* Refactoring the code to follow modern web standards.
* Improving responsiveness for all device types.
* Adding new sections as my portfolio grows.
